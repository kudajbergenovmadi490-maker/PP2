# Practice 7 – Pygame Games

Three classic games built with Pygame.

## Installation
```bash
pip install -r requirements.txt
```

## Games

| Game | Folder | Description |
|------|--------|-------------|
| Mickey's Clock | `mickeys_clock/` | Analog clock using Mickey Mouse hand graphics |
| Music Player | `music_player/` | Keyboard-controlled MP3/WAV player |
| Moving Ball | `moving_ball/` | Arrow-key-controlled red ball with boundary detection |

## Running each game
```bash
# Mickey's Clock
cd mickeys_clock && python main.py

# Music Player (put audio files in music_player/music/)
cd music_player && python main.py

# Moving Ball
cd moving_ball && python main.py
```
